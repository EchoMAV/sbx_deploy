Server Stream Setup

Input the parameters to control the video stream sent to the cloud. Use of this stream requires the EchoLite to have an internet connection. For LTE, it is recommended to use the lowest bitrate which provides acceptable quality. The Stream Name is a unique ID for the video stream, typically set to the aircraft's serial number.
